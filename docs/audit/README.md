# Audit artifacts (local)

Este directorio contiene material de auditoría/diagnóstico generado en local.
No debe incluir artefactos pesados de runs (HDF5/NPZ/etc.) salvo decisión explícita.

- `runs_audit_backup/`: backups de auditoría (manifests, summaries, logs).
- `malda/`: material auxiliar de análisis/auditoría.
