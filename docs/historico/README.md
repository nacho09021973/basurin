# Documentación histórica

Este índice reúne documentos que se conservan por trazabilidad técnica, pero que ya no deben actuar como onboarding operativo del repositorio.

## Criterio de degradación

Un documento pasa a histórico cuando cumple al menos una de estas condiciones:

- describe una interfaz o flujo ya sustituido por wrappers BRUNETE
- depende de artefactos o lanzadores antiguos que ya no son la entrada pública recomendada
- mezcla estado puntual de una fecha concreta con documentación operativa general
- ya no refleja el estado operativo vigente del repositorio

## Regla de uso

Si el objetivo es ejecutar o interpretar el sistema hoy, usa la documentación en `docs/operativo/` y el manual de BRUNETE.
Si el objetivo es reconstruir historia de decisiones, auditorías o flujos retirados, usa este índice histórico.
